TYPES OF CYBER SECURITY ?

It is categorized five distinct types:

  • 1. CRITICAL INFRASTRUCTURE SECURITY
  • 2. APPLICATION SECURITY
  • 3. NETWORK SECURITY
  • 4. CLOUD SECURITY
  • 5. INTERNET OF THINGS(IoT) SECURITY



CRITICAL INFRASTRUCTURE SECURITY

It consists of the cyber physical system that modern societies rely on.
for ex:-

  • Electricity grid.
  • Water Purification.
  • Traffic Lights.
  • Shopping centers.
  • Hospitals.



APPLICATION SECURITY

Application security uses software and hardware methods to tackle external threats than can arise in the development stage of an application. It is security as one of the several must have security measures adopted to protect your system.
-:- Types of security-:-

  • Antivirus Programs.
  • Firewalls.
  • Encryption Programs.



NETWORK SECURITY

Network security is a practice of securing a computer network from intruders, whether targeted attackers or opportunistic malware.
-:- Network Security Basics-:-

  • Protect.

    (correct configuration of your systems and network.)

  • Detect.

    (identify any configuration changes or when network traffic indicates an issue.)

  • React.

    (quickly respond to the flagged issues and make it a priority to get the network secured and back to normal as quickly as possible.)



CLOUD SECURITY

Cloud security is a software based security tool that protects and monitors the data in cloud resources.

"CLOUD SECURITY IS CONCERNED WITH SECURING DATA APPLICATION AND INFRASTRUCTURE IN THE CLOUD"

-:- cloud security advantages-:-

  • Identifying and Access Management.
  • Securing data in cloud.
  • Securing the operating System.
  • Managing security monitoring, Altering, Audit, Trail and incident Response.



INTERNET OF THINGS(IoT) SECURITY

IoT involves securing smart devices and networks that are connected over the internet.
for ex:-

  • Smart Phones.
  • Laptops.
  • Digital watches.
  • Iot devices i.e AC, door locks, fitness trackers etc.
